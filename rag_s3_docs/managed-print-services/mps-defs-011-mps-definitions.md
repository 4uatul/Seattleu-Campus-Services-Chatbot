# MPS Definitions

**Section:** Managed Print Services
**Breadcrumb:** Campus Services > MPS > Definitions
**Source URL:** https://redhawks.sharepoint.com/sites/Intranet-CampusServices/SitePages/Definitions.aspx
**Tags:** MPS, definitions, glossary, print group, MFD

## Content

MPS PRINT GROUP: Outlook address book group linked to activity string for printing charges. Used by faculty/staff/student employees for work-related printing. PRINT GROUP MEMBER: user with group membership who can select group for self-serve printing charges. PRINT ADMINISTRATOR: manages print group membership; also Usage Auditor. USAGE EMAIL DISTRIBUTION LIST: links auditors to activity strings for monthly usage memos (MPS and postage). USAGE AUDITOR: receives usage memos monthly, granted Data Hub subsite access. MPS PRINTER: nearly all campus printing devices including Sharp MFDs and HP desktops. MPS printers labeled with asset tag GS followed by 3 numbers. MFD/MFP: Multi-Function Device/Printer—printing, copying, scanning. NON-STANDARD USER: not faculty/staff/student needing printing access—guests, vendors, volunteers, temporary staff, emeritus faculty, student employee SE accounts.
