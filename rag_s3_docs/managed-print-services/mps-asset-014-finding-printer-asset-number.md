# Finding Printer Asset Number

**Section:** Managed Print Services
**Breadcrumb:** Campus Services > MPS > Asset Numbers
**Source URL:** https://redhawks.sharepoint.com/sites/Intranet-CampusServices/SitePages/Finding-the-Asset-Number-of-a-Printer.aspx
**Tags:** MPS, asset number, printer, identification

## Content

Asset Number/Equipment ID uniquely identifies MPS printer for service requests. FIRST HILL: 5-digit code with 2 letters + 3 numbers. Most begin with GS, some with HD or KC. At least one label on front of device. SLU/CORNISH: 7-digit code beginning with KC + 5 numbers. If cannot find, may not be MPS printer or removed—provide specific location.
