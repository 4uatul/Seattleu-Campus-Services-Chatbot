# MPS Device Move/Add/Change

**Section:** Managed Print Services
**Breadcrumb:** Campus Services > MPS > Device Move/Add/Change

**Tags:** MPS, move, add, change, device, printer

## Content

Only MPS can move devices (copiers and printers). Form needed any time adding, removing, or moving MPS printer—including within same room. Device types: Sharp multi-function devices (MFDs), HP single-function desktop printers (SFPs). By contract, printers can only be moved by MPS team or vendor. Changing data port causes printer to stop working—ports must be provisioned. INSTRUCTIONS: Provide data port number or photo of port labels. Submit form at Move/Add/Remove MPS Printer via IT Service Desk Jira Service Management. PRINTER LOCATIONS DETERMINED BY: Monthly output volumes, number of users, ADA compliance, fire code compliance, data/power availability, ventilation, safety concerns, device availability. PROCESSING TIME: Minimum 5 business days. Moving MFDs between buildings takes longer (vendor schedules truck/movers).
