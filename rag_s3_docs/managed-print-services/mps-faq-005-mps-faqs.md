## Content

Categories: Payments and Costs, Paper, Troubleshooting, Printer Move/Add/Change. DOCUMENT READINESS: Set size to 8.5x11 (letter) or scalable. Margin of at least 0.25 inches. Use fit option in Adobe Acrobat. Open PDFs in Word to alter configurations. WEB PRINT: Option for simple jobs without PaperCut client. Feature-limited—no simplex/grayscale/color/scaling/stapling options. For full features, print from university computer, properly set-up personal computer, or iOS/Android connected to su-print-queue. PRINT DEFAULTS: Applied by print queue after receiving job. Settings changed in Word/Acrobat/browser are NOT respected. Must change in printer Properties dialogue box. DUPLEX: Fall 2019 University set duplex (double-sided) as default for eco-friendly printing.
