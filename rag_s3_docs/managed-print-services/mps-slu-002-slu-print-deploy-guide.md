**Section:** Managed Print Services

## Content

Print Deploy available for ITS-managed workstations. STEPS: 1) Locate tray icon (right corner, green icon, click up arrow if not visible), 2) Click icon, select View my Printers, 3) View Print Deploy Client showing current printers; click Add printers, 4) Choose printer. Find-Me-Xerox is recommended queue—allows release at any Xerox printer on campus. Brother printer = select its queue. Printer naming format: \\mccprint2\MCC_7F_COLOR (mccprint2 = print server, MCC = building, 7F = floor, COLOR = function). Help: IT Service Desk Help Center.
