# Print Support Scope

**Section:** Managed Print Services
**Breadcrumb:** Campus Services > MPS > Support Scope
**Source URL:** https://redhawks.sharepoint.com/sites/Intranet-CampusServices/SitePages/Print-Support-Scope.aspx
**Tags:** MPS, support, scope, PDF, troubleshooting

## Content

MPS supports shared network print devices: hardware maintenance, driver installation, network connectivity, supply management. Limited to core printing functions. SUPPORTED: Printing from standard OS print dialogs (Windows, macOS), driver installation/configuration, network connectivity, hardware issues, supplies. NOT SUPPORTED: Printing within specialized design/production software (Adobe InDesign, Illustrator, Photoshop, QuarkXPress), color management/ICC profiles/soft-proofing, imposition/bleed/crop marks, application-level print dialog settings within third-party software. RECOMMENDED PRACTICE: Convert to PDF first—test by exporting/saving as PDF, print from Adobe Acrobat or built-in PDF viewer. If prints correctly as PDF, issue is with source application/file, not print system.
