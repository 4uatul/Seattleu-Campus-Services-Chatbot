# MPS Faculty/Staff Information

**Section:** Managed Print Services
**Breadcrumb:** Campus Services > MPS > More Info > Faculty/Staff
**Source URL:** https://redhawks.sharepoint.com/sites/Intranet-CampusServices/SitePages/Faculty-Staff-Information.aspx
**Tags:** MPS, faculty, staff, print group, driver

## Content

To use activity budget lines for printing: 1) Must be ADDED TO MPS PRINT GROUP. Student workers may be added. 2) Must have PRINTER DRIVER AND APPLICATION INSTALLED. Mac users: usually preinstalled, contact IT Service Desk if not. PC users: restart university computer—driver auto-installs on login. PC laptops: plug into University Network via ethernet, restart. PERSONAL PRINTING: Faculty/staff can use MPS hardware—select personal account instead of Print Group. Must have funds in One Card account.
