# Supplementary ID Badges

## Content

Supplementary ID Badges serve as official credentials for personnel performing specialized University functions. Provide visible verification of professional role. NOT FOR: physical security (building/room access), managed services (printing/library), financial transactions (campus payment/meal plans). For those features, consider Privilege Card. HOW TO GET: Available through ProcureSU. Visit Privilege Cards/Supplementary ID Badges page. Manual entry for single individual or Excel upload for multiple. All fields required except Preferred Name. Designs: Generic, HRL, Facilities, ITS, HES. Custom requests: PrintandMail@seattleu.edu.
